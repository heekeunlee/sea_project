# Ocean Explorer - 2026 바다꾸러기 키움사업

바다의 아름다움과 중요성을 배우고, 해양 생태계를 지키기 위한 개인 활동을 기록하는 프로젝트형 웹 애플리케이션입니다.

## 🌊 주요 기능
1. **🏠 홈**: 프로젝트 목표와 배경 설명
2. **📚 학습**: 해양 생태계 및 오염, 기후 변화 교육 콘텐츠
3. **💡 퀴즈**: 배운 내용을 확인하는 대화형 퀴즈
4. **📷 활동 기록**: 직접 시도한 바다 보호 활동(비치코밍, 분리수거 등) 사진 기록
5. **🗺️ 바다 지도**: 국내외 주요 해양 보호구역 시각화

## 🛠️ 기술 스택
- **Language**: Python
- **Framework**: Streamlit
- **Visualization**: Plotly
- **Data Handling**: Pandas

## 🚀 시작하기
```bash
pip install -r requirements.txt
streamlit run app.py
```
